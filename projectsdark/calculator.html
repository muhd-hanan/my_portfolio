Calculator

This project is a simple and responsive web-based calculator built with HTML, Tailwind CSS, and JavaScript that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, division, and decimal calculations; it features a clean modern UI, a clear button to reset the display, an equals button to show results instantly, and a fully responsive design that works smoothly across mobile, tablet, and desktop devices, making it a lightweight yet practical example of combining stylish frontend design with functional JavaScript logic.

Technologies

  • HTML5, CSS3, JavaScript, Tailwind CSS
