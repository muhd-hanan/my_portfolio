Link tracking System

I’m building Link Tracking, a web application designed to generate, share, and monitor custom links, making it easy to track engagement on personal portfolios, resumes, GitHub profiles, or any external websites. The system allows users to paste a URL, generate a shortened tracking link, and then share it with others while collecting insights on how many times the link has been opened. Each user has their own dashboard to view the performance of their generated links, while the Admin Panel provides deeper insights into the overall usage of the platform, including the total number of registered users, how many links have been generated, and how much traffic has been received. Built with a powerful tech stack — Django (backend), PostgreSQL (database), TailwindCSS (responsive UI), and JavaScript (interactivity) — the platform combines clean design with scalable performance. The purpose of this project is to help individuals like me track how effectively their portfolios, resumes, and professional profiles are being accessed, turning simple link sharing into actionable data.

Technologies

  • Backend: Python, Django Framework
  • Frontend: HTML5, CSS3, JavaScript, Tailwind CSS
  • Database: PostgreSQL
