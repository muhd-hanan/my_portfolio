Meme generator

This Meme Generator allows users to create personalized memes with ease. You can upload your own image or load a random meme fetched from an online API. The generator supports three types of text: top, middle, and bottom. The top and bottom texts automatically appear at the top and bottom of the meme image, while the middle texts are fully customizable: you can add multiple middle texts, drag them to any position, select a text by clicking it, and delete the selected text using the delete button. Users can also change the color of all texts using the color palette. Once satisfied, the meme can be downloaded as a PNG image with the exact positions, colors, and styles preserved. The interface is simple, interactive, and visually appealing, making it easy to generate memes with multiple draggable texts without affecting the top or bottom text positions.

Technologies

  • Backend: Python, Django Framework
  • Frontend: HTML5, CSS3, JavaScript, Tailwind CSS
